개인정보처리방침

앱 이름: 프로야구 클립뷰어 (비공식)
최종 업데이트: 2026-06-24

본 방침은 "프로야구 클립뷰어"(이하 "본 앱")가 이용자의 정보를 어떻게 다루는지 설명합니다. 본 앱은 KBO, 각 구단, YouTube, Google, 네이버와 무관한 비공식 앱입니다.

1. 수집하는 개인정보

본 앱은 회원가입·로그인 없이 동작하며, 개발자가 이름·이메일·전화번호 등 직접적인 개인 식별 정보를 요구하거나 수집·저장하지 않습니다.

  • 회원가입·로그인·계정 연동이 없습니다.
  • 이름, 이메일, 전화번호 등 개인 식별 정보를 요구하지 않습니다.
  • 단, 아래 4항의 광고(Google AdMob)를 통해 광고 식별자(Advertising ID), 기기 정보, 대략적 위치(IP 기반) 등 일부 정보가 자동으로 수집·처리될 수 있습니다.

2. 기기에만 저장되는 정보

  • 즐겨찾기 / 내 팀 설정: 기기 내부 저장소(Android SharedPreferences)에만 저장되며 외부로 전송되지 않습니다. 앱을 삭제하면 함께 삭제됩니다.

3. 제3자 서비스

본 앱은 콘텐츠 제공을 위해 다음 외부 서비스를 호출합니다. 이때 통신 과정에서 검색어, 기기의 IP 주소·기본 정보 등이 해당 서비스로 전달될 수 있으며, 이는 각 서비스의 정책을 따릅니다.

  • YouTube Data API / YouTube 임베드 플레이어 (Google LLC): 영상 검색·정보 조회 및 재생에 사용됩니다. 본 앱은 YouTube API Services 를 이용합니다.
    • Google 개인정보처리방침: https://policies.google.com/privacy
    • YouTube 서비스 약관: https://www.youtube.com/t/terms
  • 네이버 스포츠 일정 API (네이버): 경기 일정·점수·순위 정보를 표시하는 데 사용됩니다.

본 앱 개발자는 위 서비스에 전달되는 데이터를 별도로 수집하거나 저장하지 않습니다.

4. 광고 (Google AdMob)

본 앱은 무료 제공을 위해 Google 의 모바일 광고 서비스 Google AdMob을 사용하여 배너 및 전면 광고를 표시합니다. 이 과정에서 AdMob(Google LLC)은 광고 제공·측정·맞춤화를 위해 다음과 같은 정보를 자동으로 수집·처리할 수 있습니다.

  • 광고 식별자(Advertising ID), 기기 정보(모델·OS 버전 등), IP 주소 기반 대략적 위치
  • 앱 내 광고 상호작용(노출·클릭 등) 정보

수집된 정보는 Google 의 광고 정책에 따라 처리됩니다: https://policies.google.com/technologies/ads

이용자는 기기 설정에서 광고 ID 재설정 또는 맞춤 광고 선택 해제를 할 수 있습니다 (Android: 설정 → Google → 광고).

5. 아동의 개인정보

본 앱은 만 14세 미만 아동을 주 대상으로 하지 않으며, 아동의 개인정보를 고의로 수집하지 않습니다.

6. 보안

본 앱 개발자는 이용자의 개인정보를 저장하는 별도의 서버나 데이터베이스를 운영하지 않습니다. 앱과 외부 서비스(광고 포함) 간 통신은 HTTPS 를 통해 이루어집니다.

7. 방침의 변경

본 방침은 관련 법령이나 서비스 변경에 따라 수정될 수 있으며, 변경 시 본 문서에 갱신일과 함께 게시합니다.

8. 데이터 삭제 요청

본 앱은 회원가입·로그인이 없어 개발자가 보관하는 개인정보가 없습니다. 사용자는 다음 방법으로 데이터를 삭제할 수 있습니다.

  • 기기 내 저장 데이터(내 팀·즐겨찾기 등): 앱을 삭제하면 기기에 저장된 모든 데이터가 즉시 함께 삭제됩니다.
  • 광고 식별자: Android 설정 → 개인정보 보호 → 광고에서 광고 ID를 재설정하거나 삭제할 수 있습니다.
  • 기타 삭제 요청: 아래 이메일로 데이터 삭제를 요청하실 수 있으며, 확인 후 처리합니다.
